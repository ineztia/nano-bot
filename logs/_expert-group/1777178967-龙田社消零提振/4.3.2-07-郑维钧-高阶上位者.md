# 4.3.2-07 郑维钧·高阶上位者（原坪山区经济分管领导）

**从区级视角看龙田——我能给你什么、你不能犯什么错**

## 一、区级态度：看排名、看增量、看合规

区级是愿意支持龙田提升社消零的——龙田的增长就是全区的增长。关键是怎么"报上去"。

## 二、区级可以给龙田的资源

| 资源 | 争取难度 | 关键路径 |
|------|----------|----------|
| 社消零专项政策包（试点） | 中 | 街道党工委书记向分管副区长汇报 |
| 统计口径协调（过渡期灵活） | 低 | 经服中心对接区统计局 |
| 跨区协调 | 高 | 需区长层面出面 |
| 产业空间支持 | 中 | 结合区级"工改工"项目 |
| 招商资源倾斜 | 低 | 对接区投资推广署 |
| 考核加分 | 低 | 区发改/统计确认 |

## 三、区级的三条红线

1. **不可虚假注册**——企业必须有实际经营
2. **不可跨口径调数**——批发不可硬说零售
3. **不可恶性抢夺**——新注册归龙田，不抢邻街存量

## 四、给龙田的具体建议

1. **打"试点"牌**：包装为"坪山区制造业街道社消零提升改革试点"
2. **先见效果再要资源**：用3个月拿出初步成果，再争取区级资源
3. **与统计部门做盟友**：主动报告"应统未统"，请其指导合规纳统
4. **把握时间窗口**：年中、年末两次数据审核窗口前完成纳统
