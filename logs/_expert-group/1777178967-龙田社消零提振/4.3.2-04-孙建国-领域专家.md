# 4.3.2-04 孙建国·领域专家（街道经济治理与招商引资）

**街道权限内可用的政策工具箱——能做什么、不能做什么、怎么争取更多**

## 一、街道办经济工作的"权限地图"

### 街道可直接使用的工具

| 工具 | 说明 | 对社消零的作用 |
|------|------|---------------|
| 企业走访与动员 | 街道对辖区内企业有走访联系职责 | 排查应统未统、推动企业纳统配合 |
| 企业服务窗口 | 经服中心可为企业提供行政事务代办 | 为企业注册、变更、纳统提供便利 |
| 协调区级部门 | 街道可向上协调区发改、工信、统计等部门 | 争取政策资源、解决跨部门问题 |
| 场地资源协调 | 街道掌握部分集体物业、产业空间 | 为入驻电商企业提供低成本办公空间 |
| 社区网格力量 | 社区网格员可深入每栋楼宇摸排 | 排查隐匿电商企业、个体工商户 |
| 街道级专项资金 | 部分街道设有经济发展专项资金 | 对纳统企业给予小额奖励 |
| 荣誉与品牌 | 街道可评选"优秀企业"等 | 为企业提供社会认可，间接激励 |
| 营商环境优化 | 简化办事流程、主动上门服务 | 提升企业满意度，增加配合意愿 |

### 街道不可直接使用但可争取的工具

| 工具 | 归属 | 争取路径 |
|------|------|----------|
| 税收返还 | 区级财政 | 通过街道党工委书记向区领导汇报争取 |
| 产业扶持资金 | 区工信/发改 | 以街道名义向区级申报专项 |
| 人才房配租 | 区住建 | 协调区住建局分配名额 |
| 用地性质变更 | 区规自局 | 撰写专题报告申请 |
| 纳统标准灵活执行 | 区统计局 | 与区统计局沟通口径 |

## 二、招商引资——街道的四种路径

1. **内生型招商**：从现有规上制造业企业中挖掘商贸增量
2. **朋友圈招商**：以商引商，利用现有企业的商业网络
3. **载体招商**：用好存量空间，梳理可用产业空间制作招商手册
4. **政策招商**：争取区级为龙田定制专项政策包

## 三、街道经济工作的"隐形资源"

- 领导重视：主要领导亲自抓比任何政策都有效
- 部门联动：经服中心+统计站+市场监管所+税务所四方联动
- 集体股份公司：社区股份公司掌握大量物业
- 行业协会：牵头成立"龙田商贸企业联谊会"
- 媒体宣传：街道微信公众号、社区宣传栏免费引流

## 四、时间线建议

| 时间 | 动作 | 责任主体 |
|------|------|----------|
| 第1个月 | 成立"社消零提升工作专班" | 街道办 |

| 第2-3个月 | 向区委区政府提交专项政策申请 | 街道党工委 |
| 第2-4个月 | 启动电商虚拟园区+MCN招商 | 经服中心 |
| 第3-6个月 | 推动首批10家企业设立/回迁销售公司 | 经服中心+企业 |
| 第4-8个月 | 电商企业纳统服务集中推进 | 统计站 |
| 第6-9个月 | 区级专项政策落地 | 区级部门 |
| 第9-12个月 | 查漏补缺、冲刺全年数据 | 全专班 |
